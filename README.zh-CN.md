# AI 虚拟鼠标与交互式手势生态系统 🚀

[English](README.md) · 简体中文 · [日本語](README.ja.md)

这是一套先进的多模态人机交互（HCI）工具，可将**计算机视觉手部跟踪**、**智能手机运动传感器**和**语音命令**转化为对操作系统的实时控制、交互式 3D 模拟以及游戏化浏览器服务器。

由 **[Vishal P](https://github.com/vishalpatil133-cmd)** 开发（*11 年级学生开发者及电子与通信工程爱好者*）。

---

## 📽️ 项目概览
本项目汇集了 4 个充满未来感的实用工具，无需昂贵硬件即可打通人与机器之间的交互：

1. **AI 摄像头虚拟鼠标**（`virtual_mouse.py`）：使用普通摄像头控制光标、点击、滚动、在空中绘画，并通过语音听写文本。
2. **智能手机 Wi-Fi 空中鼠标**（`wifi_mouse.py`）：通过本地 Wi-Fi，将手机的陀螺仪和加速度计变成空中鼠标及多点触控板。
3. **赛博朋克霓虹赛车**（`car_server.py`）：用双手模拟方向盘的手势控制浏览器中的 3D 赛车。
4. **Stark 3D 全息 HUD**（`stark_server.py`）：使用缩放和平移手势操控 3D 线框方舟反应堆，并支持面向全息塑料金字塔的四向镜像投影。

---

## 🌟 主要功能与工程亮点

### ⚡ 1. 映射到指关节的稳定点击跟踪
大多数手部跟踪系统会将光标映射到**食指指尖**（`Landmark 8`）。然而，当食指弯曲以执行点击或捏合动作时，指尖会向下滑动，导致光标偏离目标按钮。
* **我们的解决方案**：将光标跟踪映射到**食指掌指关节（根部指关节）**（`Landmark 5`）。由于手指弯曲时根部指关节保持完全静止，点击可实现 **100% 稳定**，光标不会漂移！

### 🏎️ 2. 丝滑流畅的 30 FPS 性能
* 优化 MediaPipe Hands 流程，使其能以稳定的 30 FPS 运行。
* 将 `model_complexity` 降为 `0`，并在鼠标模式中把跟踪限制为 `max_num_hands=1`，从而彻底消除普通笔记本电脑上的 CPU 热降频问题。
* 集成动态 LERP（线性插值）平滑处理，让光标移动更加柔和流畅。

### 📌 3. 全局窗口快捷键
在玩游戏或使用 Photoshop 工作时，需要让摄像头预览窗口始终可见？
* **`Ctrl + Alt + A`**：将摄像头画面窗口设为**始终置顶**。操作时会发出一声高音提示音。
* **`Ctrl + Alt + V`**：将窗口恢复为普通状态。操作时会发出一声低音提示音。

## ⚙️ 安装与设置

### 1. 前置条件
确保已安装 **Python 3.8 或更高版本**。安装全部依赖：
```bash
pip install opencv-python mediapipe pyautogui numpy websockets keyboard
```
*（注意：在 Windows 上，请以管理员身份运行终端，以启用 `keyboard` 库的全局快捷键。）*

### 2. 运行实用工具

#### 🖱️ 模块 A：摄像头虚拟鼠标
```bash
python virtual_mouse.py
```
* **性能模式**：按 `H` 隐藏手部关键点和叠加图形，以减少 CPU 渲染开销。
* **始终置顶快捷键**：按 `Ctrl+Alt+A`，使摄像头画面始终显示在其他应用之上。

#### 📱 模块 B：智能手机 Wi-Fi 控制器
```bash
python wifi_mouse.py
```
* 记下控制台中显示的 IP 地址（例如 `http://192.168.1.5:8000`）。
* 确保手机连接到同一 Wi-Fi，然后在手机浏览器（Safari/Chrome）中打开该地址。锁定手机屏幕旋转，即可使用空中陀螺仪！

#### 🏎️ 模块 C：虚拟方向盘赛车
```bash
python car_server.py
```
* 在电脑浏览器中打开 `http://localhost:8002`。
* 在摄像头前像握住方向盘一样举起双手，倾斜双手即可转向！捏合右手加速，捏合左手刹车。

#### ⚛️ 模块 D：Stark 全息 HUD
```bash
python stark_server.py
```
* 在电脑浏览器中打开 `http://localhost:8001`。
* 用一只手平移、两只手缩放，连续捏合两次即可使方舟反应堆过载。按浏览器中的 `H` 键可切换四向全息投影仪模式。
