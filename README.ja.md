# AI 仮想マウス＆インタラクティブ・ジェスチャー・エコシステム 🚀

[English](README.md) · [简体中文](README.zh-CN.md) · 日本語

**コンピュータービジョンによるハンドトラッキング**、**スマートフォンのモーションセンサー**、**音声コマンド**を、OS のリアルタイム操作、インタラクティブな 3D シミュレーション、ゲーム性のあるブラウザーサーバーへ変換する、高度なマルチモーダル・ヒューマンコンピューターインタラクション（HCI）スイートです。

**[Vishal P](https://github.com/vishalpatil133-cmd)**（*11 年生の学生開発者、電子通信工学愛好家*）が開発しました。

---

## 📽️ プロジェクト概要
このプロジェクトは、高価なハードウェアを必要とせずに人と機械の隔たりを埋める、4 つの未来的なユーティリティをまとめたものです。

1. **AI Web カメラ仮想マウス**（`virtual_mouse.py`）：一般的な Web カメラを使って、カーソル操作、クリック、スクロール、空中描画、音声によるテキスト入力を行えます。
2. **スマートフォン Wi-Fi エアマウス**（`wifi_mouse.py`）：ローカル Wi-Fi 経由で、スマートフォンのジャイロスコープと加速度センサーをエアマウスやマルチタッチトラックパッドとして利用できます。
3. **サイバーパンク・ネオンレーサー**（`car_server.py`）：両手で仮想ステアリングを握るジェスチャーを使い、ブラウザー上の 3D カーシミュレーションを操作します。
4. **Stark 3D ホログラフィック HUD**（`stark_server.py`）：ズームやパンのジェスチャーで 3D ワイヤーフレームのアーク・リアクターを操作し、ホログラム用プラスチックピラミッドに対応した 4 方向ミラー投影も利用できます。

---

## 🌟 主な機能と技術上の特長

### ⚡ 1. 指の付け根に割り当てた安定したクリック追跡
多くのハンドトラッキングシステムでは、カーソルを**人差し指の先端**（`Landmark 8`）に割り当てます。しかし、人差し指を曲げてクリックやピンチを行うと指先が下へ動き、カーソルが対象のボタンからずれてしまいます。
* **本プロジェクトの解決策**：カーソル追跡を**人差し指の MCP 関節（付け根の関節）**（`Landmark 5`）に割り当てました。指を曲げても付け根の関節は完全に静止するため、クリックは **100% 安定**し、カーソルのずれが発生しません！

### 🏎️ 2. 滑らかな 30 FPS のパフォーマンス
* MediaPipe Hands パイプラインを最適化し、安定した 30 FPS で動作するようにしました。
* `model_complexity` を `0` に下げ、マウスモードでは追跡を `max_num_hands=1` に制限しました。これにより、一般的なノート PC での CPU のサーマルスロットリングを完全に解消しました。
* 動的な LERP（線形補間）スムージングを組み込み、カーソルの動きを柔らかく滑らかにしています。

### 📌 3. グローバルウィンドウホットキー
ゲーム中や Photoshop での作業中にも、Web カメラのプレビューウィンドウを常に表示しておきたい場合に利用できます。
* **`Ctrl + Alt + A`**：カメラ映像ウィンドウを**常に手前に表示**します。高い確認音が鳴ります。
* **`Ctrl + Alt + V`**：ウィンドウを通常の状態に戻します。低い確認音が鳴ります。

## 🖐️ ジェスチャーコマンド一覧

| ジェスチャー | 操作 | 説明 |
| :--- | :--- | :--- |

## ⚙️ インストールとセットアップ

### 1. 前提条件
**Python 3.8 以降**がインストールされていることを確認し、すべての依存関係をインストールします。
```bash
pip install opencv-python mediapipe pyautogui numpy websockets keyboard
```
*（注：Windows で `keyboard` ライブラリのグローバルホットキーを有効にするには、管理者としてターミナルを実行してください。）*

### 2. ユーティリティの実行

#### 🖱️ モジュール A：Web カメラ仮想マウス
```bash
python virtual_mouse.py
```
* **パフォーマンスモード**：`H` を押すとランドマークとオーバーレイグラフィックを非表示にし、CPU の描画負荷を抑えます。
* **常に手前に表示するホットキー**：`Ctrl+Alt+A` を押すと、カメラ映像がほかのアプリより前面に表示されます。

#### 📱 モジュール B：スマートフォン Wi-Fi コントローラー
```bash
python wifi_mouse.py
```
* コンソールに表示される IP アドレス（例：`http://192.168.1.5:8000`）を確認します。
* スマートフォンを同じ Wi-Fi に接続し、ブラウザー（Safari/Chrome）でこの URL を開きます。端末の画面回転をロックすると、エアジャイロスコープとして利用できます！

#### 🏎️ モジュール C：仮想ステアリングレーサー
```bash
python car_server.py
```
* PC のブラウザーで `http://localhost:8002` を開きます。
* カメラの前で仮想ステアリングを握るように両手を構え、傾けて操舵します。右手をピンチすると加速し、左手をピンチするとブレーキがかかります。

#### ⚛️ モジュール D：Stark ホログラム HUD
```bash
python stark_server.py
```
* PC のブラウザーで `http://localhost:8001` を開きます。
* 片手でパン、両手でズームし、ダブルピンチするとアーク・リアクターがオーバーチャージされます。ブラウザーで `H` を押すと、4 方向ホログラフィックプロジェクターモードを切り替えられます。
